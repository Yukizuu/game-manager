# アセットルール

## ドット絵のサイズ

ドット絵は原則として `32x32` ピクセルで作成します。

小物、アイコン、簡易エフェクトなどの小さい素材は `16x16` ピクセルで作成します。

## ファイル形式

画像アセットは透過PNGで管理します。

背景が不要なキャラクター、アイテム、UI素材は、透明背景のPNGとして保存します。

## スプライトシート

スプライトシートを追加する場合は、対応するREADMEで内容を説明します。

READMEには以下を記載します。

- スプライトシートの用途
- 1コマのサイズ
- 行と列の構成
- アニメーション順序
- 使用するゲームまたは画面
